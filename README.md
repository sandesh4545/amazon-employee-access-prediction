# amazon-employee-access-prediction
In large organizations like Amazon, employees request access to various resources. Manually approving access can be time-consuming and error-prone. This project builds a machine learning model to predict access approval based on historical employee data.
# Amazon Employee Access Prediction

## 📌 Project Overview

This project aims to predict whether an employee should be granted access to a specific resource based on their role and organizational attributes. The goal is to automate access control decisions using machine learning and reduce manual approval efforts.

## 🎯 Objective

Build a classification model that predicts employee access approval using historical data and improve decision-making efficiency.

## 📊 Dataset Features

* Role ID
* Department
* Manager ID
* Resource ID
* Role Title
* Role Family
* Role Code
* Access (Target Variable)
